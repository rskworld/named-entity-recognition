About This Dataset

This Named Entity Recognition dataset contains labeled entities including:

  • Persons: Names of people
  • Organizations: Companies, institutions, and organizations
  • Locations: Cities, countries, and geographical locations
  • Dates: Time expressions and dates
  • Money: Monetary values
  • Percent: Percentage values

The dataset is available in multiple formats (CSV, JSON, BIO) and is ready for training NER models using spaCy, NLTK, or Transformers.
