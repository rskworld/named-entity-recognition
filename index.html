Key Features

Everything you need for Named Entity Recognition projects

Labeled Entities

Pre-labeled entities including persons, organizations, locations, dates, and more for immediate use in your projects.

Multiple Entity Types

Support for PERSON, ORG, LOC, DATE, MONEY, and PERCENT entity types with comprehensive coverage.

BIO Tagging Format

Includes BIO (Beginning-Inside-Outside) tagging format ready for sequence labeling models.

Training & Test Sets

Pre-split training and test datasets for immediate model training and evaluation.

Ready for NER Models

Compatible with spaCy, NLTK, Transformers, and other popular NLP frameworks.

Multiple Formats

Available in CSV, JSON, and BIO formats for maximum flexibility and compatibility.

Batch Processing

Advanced batch processing capabilities with parallel processing support for large datasets.

Advanced Analytics

Comprehensive statistics and analytics tools for dataset analysis and quality assessment.

REST API

Built-in Flask API server for programmatic access to dataset and entity extraction.
